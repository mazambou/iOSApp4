# Family Finance Canada

## Overview

Family Finance Canada is a simple SwiftUI application designed to help Canadian families manage their monthly finances.

The application allows users to record income and expenses, monitor their monthly budget, and visualize their spending using modern SwiftUI components.

This project was developed as part of the **iOSApp4** assignment to explore new features from the **SwiftUI Cookbook**.

---

## Features

- Dashboard displaying monthly income, expenses, and remaining budget.
- Add income and expense transactions.
- View all financial transactions.
- Search transactions by name or category.
- Display monthly budget progress.
- View expense statistics by category.

---

## SwiftUI Cookbook Features Implemented

### 1. Searchable
Allows users to search transactions by title or category.

### 2. Swipe Actions
Swipe left on a transaction to edit or delete it.

### 3. Context Menu
Long press a transaction to display additional actions such as Edit, Duplicate, and Delete.

### 4. DatePicker
Select the transaction date when adding a new income or expense.

### 5. ProgressView
Displays the percentage of the monthly budget already used.

### 6. Charts
Visualizes expenses by category using Apple's Swift Charts framework.
